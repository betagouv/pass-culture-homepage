Devenez ambassadeur du pass Culture !

Remplissez des missions ! 😲 

Vous vous sentez capable d’expliquer le pass Culture dans votre fac ou votre lycée, de distribuer des flyers, de faire des publications Instagram pour le pass Culture ? Alors rejoignez les ambassadeurs du pass Culture !

Vos missions, si vous les acceptez sont les suivantes :

👉 Distribuez un lot de tracts dans un établissement (lycée, association, mission locale)
👉 Partagez le lien de parrainage à 10 personnes
👉 Publiez une activité du pass Culture à laquelle vous avez participé sur les réseaux sociaux
👉 Partagez le lien du pass Culture sur un groupe (promo, association)
👉 Contactez un acteur culturel pour qu’il publie des offres sur le pass
👉 Faites un partenariat avec une association (ex: BDE), qui communique sur le pass Culture
👉 Présentez le pass Culture dans un établissement (lycée, fac, mission locale)
👉 Présentez-vous comme ambassadeur du pass Culture dans une vidéo portrait chinois
👉 Rencontrez les services culturel ou jeunesse de votre commune pour parler du pass Culture
👉 Tenez un stand pass Culture à la fac, au lycée, dans un forum
